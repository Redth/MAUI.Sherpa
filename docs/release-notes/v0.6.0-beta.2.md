## 🏔️ v0.6.0-beta.2 — "Fixing the Ropes"

You don't just clip into the fixed lines and trust them blindly — you check every anchor, re-tie every knot, and make sure the route is solid before moving higher. This release is that careful rope-check: the inspector's coordinate system has been completely rebuilt, leaking processes have been plugged, and the UI has been tightened up for a smoother ascent.

### ✨ Improvements

- **WindowBounds Migration** — The inspector's coordinate system has been rebuilt from the ground up, switching from heuristic relative-bounds accumulation to absolute `windowBounds` from MauiDevFlow. Hit testing, overlay drawing, and DPI calculation are all simpler and more reliable now.
- **Synthetic Element Support** — NavBarTitle, FlyoutButton, ShellTab, and other synthetic elements now show up properly in hit testing and get overlay highlights just like regular MAUI views.
- **Screenshot Click Accuracy** — Switched from unreliable `offsetX/Y` (which varied depending on which DOM element caught the click) to `clientX/Y` minus the image bounding rect for rock-solid coordinate mapping.
- **Screenshot Panel Polish** — The screenshot now has a subtle shadow, fits within the panel while preserving aspect ratio (no more scrollbars), and a draggable splitter lets you resize the screenshot vs. tree panes.
- **Tree ↔ Screenshot Sync** — Refreshing the visual tree now also refreshes the screenshot, and vice versa — no more stale views after changes.
- **Hit Test Debug Toggle** — A bug icon toggle shows the full coordinate chain for each click, with a copy button for easy sharing when diagnosing issues.
- **Native Toolbar on Inspector Pages** — Both the DevFlow dashboard and inspector pages now set their own native macOS toolbar items instead of inheriting stale buttons from previously visited pages.

### 🐛 Bug Fixes

- **xcdevice Process Leak** — `xcdevice list` and `xcrun simctl list` processes were surviving app exits and accumulating over time. All spawned processes now have cancellation-triggered kills and a 10-second hard timeout.
- **DPI Miscalculation with Modals** — Opening a modal page caused the DPI to spike to absurd values because `FindFirstWindowBounds` was returning a tiny synthetic element. Replaced with `FindLargestWindowBounds` to always get the actual window dimensions.
- **Overlay Flashing Away** — Selecting a synthetic element via hit test would briefly show the overlay, then lose it when `GetElementAsync` replaced the element with a fresh object missing the enriched bounds. Bounds are now preserved across detail fetches.

### 🔧 Infrastructure

- **MauiDevFlow 0.17.5** — Updated all agent packages and the CLI tool from 0.15.0 to 0.17.5, bringing stable element IDs, improved synthetic element support, and new `windowBounds` data.
- **Copilot Dialog Rewrite** — The Copilot chat modal has been migrated from a native C# page to an all-Blazor implementation, fixing typing lag, clipping, and styling issues.
- **README Overhaul** — Updated with website screenshots, Linux support documentation, and new sections for DevFlow and Publish Profiles.
