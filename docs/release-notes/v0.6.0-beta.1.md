## 🏔️ v0.6.0-beta.1 — "Switching to the North Face"

Sometimes you're halfway up a familiar route and you spot a completely different line — steeper, more technical, but unmistakably better. That's what happened here. We were comfortably ticking off 0.5.0 beta patches when the DevFlow App Inspector feature grew into something so big it needed its own basecamp number. Think of 0.6.0 as pivoting to the north face: same mountain, dramatically more ambitious route.

### 🆕 New Features

- **DevFlow App Inspector** — A full remote inspector for running .NET MAUI apps. Browse the visual tree, tap elements, read and edit properties live, take in-process screenshots, and monitor network requests — all without touching the running device. The biggest single feature since launch. (#82)
- **Visual Tree with VS Code–style Guides** — The inspector renders a collapsible tree with indent guide lines, per-node collapse-all, and direct screenshot-locate buttons. Click a tree node to highlight it on the live screenshot overlay.
- **Interactive Screenshot Inspect Mode** — Click anywhere on the app screenshot to identify the element under it via server-side hit testing. A crosshair toggle, highlight overlay, and element details panel make it feel like a proper dev tools experience.
- **Type-Aware Property Editors** — The properties panel includes inline editors for colors, thicknesses, corner radii, brushes (including a draggable gradient stop editor), booleans, enums, and text — all applying changes live to the running app.
- **Network Inspector** — A DataGrid-based network tab captures HTTP requests and responses with resizable columns, sorting, filtering, and syntax-highlighted request/response bodies.
- **WebView / CDP Inspector** — Browse Blazor WebView DOM snapshots, evaluate JavaScript, view page source with syntax highlighting, and select from multiple WebView targets.
- **Application Log Viewer** — Filterable log viewer for native `ILogger` and WebView `console.*` output with level/source/search filtering and clear-per-tab support.
- **WebSocket Log Streaming** — Logs now stream in real-time over WebSocket instead of polling, with automatic reconnection (exponential backoff) and graceful fallback to REST polling for older agents.
- **Vaultwarden / Bitwarden Cloud Secrets** — A new cloud secrets provider backed by Vaultwarden's REST API, so you can sync keystores and signing secrets through your own self-hosted vault. (#59)
- **GitHub Pages Website** — A full marketing site at the project's GitHub Pages: landing page, getting-started guide, DevFlow documentation, and a dark-mode screenshot gallery. Mountain divider SVG included.

### ✨ Improvements

- **Platform-Colored Agent Cards** — DevFlow agent cards now show platform-specific squircle icons with brand color overlays (Apple grey, Android green, Windows blue, Linux orange) and device-type indicators.
- **Demo Mode Blur** — Sensitive Apple data (identity names, key IDs, certificate names, profile names, device names, bundle IDs) is now blurred across all pages and dialogs when Demo Mode is enabled in Settings.
- **Inline Modals → Native Hybrid Modals** — Publisher dialog, Apple config wizard, and secret picker migrated from Blazor inline modals to native platform sheets for a more polished feel.
- **Multi-Window & Multi-WebView Support** — The visual tree inspector supports multiple app windows and multiple Blazor WebView targets within a single connected agent.
- **Agent Version Enforcement** — The inspector displays the connected agent's version and blocks interaction if the agent is below the minimum supported version (currently 0.16.0).
- **Reusable DataGrid Component** — A new `DataGrid.razor` component with resizable columns, multi-column sorting, text filtering, and custom cell templates — used by the network inspector and available for future pages.

### 🐛 Bug Fixes

- **Copilot CLI Not Found in Release Builds** — Fixed the CLI binary not being located on Mac Catalyst release builds. (#81)
- **CDP Response Parsing** — Fixed JavaScript `eval` and DOM queries returning raw CDP wrapper objects instead of extracted values.
- **Android Shell Channel Crash** — Fixed the Android device shell closing prematurely when the stderr reader completed before stdout.
- **Apple Config Wizard Layout Shift** — Prevented the Clear button from causing layout jumps by using `visibility: hidden` instead of conditional rendering.
- **SecureStorage Debug Reliability** — Hardened the debug-mode fallback storage and replaced alerts with toasts for doctor fix failures.
- **Website Hero Spacing** — Fixed the nav overlapping the mascot and CTA buttons being cut off by the mountain divider SVG.

### 🔧 Infrastructure

- **MauiDevFlow 0.15.0+ Packages** — Bumped `Redth.MauiDevFlow.Agent` and `.Blazor` NuGet packages across all three app heads (Mac Catalyst, macOS AppKit, Linux GTK).
- **CI `publish_artifacts` Input** — Added a `workflow_dispatch` input to optionally skip artifact publishing during test builds.
- **GitHub Pages Workflow** — New `pages.yml` workflow deploys the website from the `website/` directory on pushes to `main`.
