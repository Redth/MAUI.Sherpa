## 🏔️ v0.7.1 — "Rechecking the Knots"

Before you leave camp and trust your gear on a steep face, you give every knot, strap, and
carabiner one more tug. `v0.7.1` is that kind of release: a new Azure DevOps secrets cache,
tighter Apple certificate handling, snappier forms, and a few important platform fixes that keep
Sherpa dependable when the trail gets fiddly.

### 🆕 New Features

- **Azure DevOps Cloud Secrets Provider** — Sherpa can now store secrets in Azure DevOps Variable
  Groups, giving teams another secure stash for publish workflows alongside the existing cloud
  backends. (#121)
- **Apple Certificate P12 Import & CSR Workflow** — Apple certificate creation now keeps the
  private key clipped to the certificate and imports the resulting P12 into local storage right
  away, making export, backup, and signing much less of a rope tangle. (#118)

### ✨ Improvements

- **Sidebar Action Bar** — Doctor, Settings, and Copilot now live in a persistent sidebar action
  bar, so the tools you reach for most are always clipped to your harness instead of buried on the
  Dashboard. (#120)
- **Live-Saving Modal Forms** — Text fields across publishing, Apple, Google, and cloud-provider
  dialogs now update on every keystroke, so Save and Submit react immediately instead of waiting
  for focus to leave the field. (#117)
- **Ready-by-Default Pickers** — Certificate, bundle ID, and device registration forms now honor
  their default picker selections from the start, so the primary action is no longer stuck at the
  trailhead. (#119)

### 🐛 Bug Fixes

- **Publish Wizard Finish Error** — Finishing a successful publish no longer trips a
  `NullReferenceException` when you click Done; the wizard now closes only after its result is
  safely in place. (#116)
- **Windows Certificate Store Imports** — Importing P12 certificates on Windows now lands in the
  native `CurrentUser\\My` store, keeping local certificate installs on the right path. (#124)
- **Intel Mac Copilot Bundling** — The AppKit build once again packs the right Copilot runtime for
  Intel Macs, so both sides of the rope team get the tooling they need. (#125)

---

*We haven't left this stretch of trail yet, but the knots are tighter, the gear is sorted, and
the next pitch looks a lot less fussy.* 🪢
