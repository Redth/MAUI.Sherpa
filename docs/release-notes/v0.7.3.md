## 🏔️ v0.7.3 — "Adjusting the Harness"

Sometimes the climb does not need a new route — it just needs the rope team to stop
turning away climbers who already have the right gear. `v0.7.3` is a small compatibility
patch for the MauiDevFlow trail: older agents are welcome back onto the line, and the
profiling panel now makes it clearer when a particular route simply is not open yet.

### 🐛 Bug Fixes

- **Older MauiDevFlow agent compatibility** — Sherpa now accepts MauiDevFlow `0.1.0`
  and prerelease-tagged agent versions, instead of blocking connections unless the agent
  reported a much newer stable build.
- **Profiler fallback on older agents** — When a connected agent does not expose the
  profiling APIs yet, the profiler tab now shows a clear explanation instead of looking
  like it is still trudging through the fog.

### ✨ Improvements

- **Consistent version parsing** — MauiDevFlow version handling is now centralized so
  prerelease and build-metadata suffixes are compared consistently and displayed more
  cleanly throughout the inspector UI.

---

*A quick strap check, a better knot, and the whole team moves more smoothly up the ridge.* 🧗
