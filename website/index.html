Features

Everything you need for MAUI development

From environment health checks to signing certificates — Sherpa handles the tedious parts so you can focus on building your app.

MAUI Doctor

Diagnose your development environment in seconds. Check .NET SDKs, workloads, dependencies, and get AI-powered fix suggestions via GitHub Copilot.

Android SDK Manager

Browse, search, and install SDK packages. Manage platform tools, build tools, and system images with a visual interface instead of command-line sdkmanager.

Emulators & Simulators

Create, start, and manage Android emulators and iOS simulators. Take snapshots, adjust settings, and launch inspectors — all from one place.

Android Keystores

Create signing keystores, view certificate signatures, export PEPK keys for Google Play, and sync keystores across machines via cloud storage.

Apple Certificates & Profiles

Create, download, and manage Apple signing certificates and provisioning profiles. Edit capabilities, register devices, and export CI secrets.

Device Inspectors

Inspect running devices with Logcat viewers, file browsers, shell terminals, and screen capture — for both Android and iOS.

DevFlow App Inspector

Inspect running MAUI apps remotely — browse the visual tree, capture screenshots, monitor network requests, and debug Blazor WebViews live.

GitHub Copilot

Chat with GitHub Copilot directly inside Sherpa. Get AI-assisted help diagnosing environment issues, with suggested prompts for common tasks.
